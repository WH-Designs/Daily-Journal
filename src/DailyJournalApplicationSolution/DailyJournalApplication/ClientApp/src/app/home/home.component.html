Welcome to my Daily Journal Application

What this site is about:

It is a daily journal application where a user can go to log their day to day and reflect on their day and how some things may have affected them or changed them in any way shape or form. The user will be able to fill out a form which has a simple user interface in which has a form in which the user can log their daily events and a log in which they can see past days and be able to reflect on them.

Extra Information:

This application is my first application using angular and I am making this application because I felt I needed a better tool to use to track my day to day life and all of the events and things that happen during it so that I can practice and be more self aware and be able to learn and grow more as a person.
